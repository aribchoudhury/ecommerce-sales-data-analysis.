E-Commerce Sales Data Analysis

Project Overview

This project focuses on analyzing a large-scale e-commerce sales dataset using Python. The objective is to clean, process, visualize, and extract meaningful business insights from sales transactions.

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

Key Findings

* December generated the highest sales revenue.
* San Francisco recorded the highest number of orders.
* Peak customer purchasing times were 11:00 AM and 7:00 PM.
* Electronics contributed the highest share of revenue.

Project Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Monthly Sales Analysis
* City-wise Order Analysis
* Advertisement Timing Analysis
* Product Category Analysis
* Business Insights and Recommendations
